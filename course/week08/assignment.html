Probability and Bayes Thinking — Assignment Pack

Use these tasks for classwork, homework, lab practice, or portfolio evidence.

Tasks

  1. Simulate 100 coin tosses and compare theoretical and experimental probability.
  2. Write Python code to compute dice probabilities.
  3. Solve one conditional probability and one Bayes theorem question with steps.

Submission checklist

  • Clean code with comments
  • Input and output samples
  • One short explanation paragraph
  • One reflection: what was easy and what was difficult

Teacher note

Encourage students to explain their reasoning. Even in beginner weeks, clarity matters as much as code correctness.
