Course-wide structure

Before opening a week, it helps to understand the bigger course logic. The first phase builds coding confidence, the middle phase builds analytical and modeling ability, and the final phase turns knowledge into working systems.

Foundation Layer

Python basics, logic, conditions, loops, functions, and data structures. This layer gives students the ability to read, write, and debug code before they touch real datasets.

Data Layer

NumPy, pandas, CSV, Excel, cleaning, filtering, summaries, charts, and data movement. Students learn how raw data becomes understandable and useful.

Modeling Layer

Statistics, regression, classification, train-test thinking, model interpretation, and cautious reasoning about what predictions do and do not mean.

Application Layer

Real datasets, guided projects, domain caution, ethics, final capstone, and deployment. Students finish by presenting something that resembles a real working solution.

Week 1: Python Foundations I

The course starts with programming confidence. Students learn the grammar of Python: variables, basic data types, input and output, arithmetic, expressions, and program flow. The goal is to remove fear and build fluency.

Week 01
Core Topic: Python Basics Difficulty: Beginner Mode: Theory + Lab

What students learn

  • Variables, strings, numbers, and simple expressions.
  • How Python reads values and prints results.
  • The meaning of operators and expression evaluation.
  • How to think in a step-by-step computational way.

Why this week matters

  • It creates the language base for the full AI-ML journey.
  • Students begin solving small but complete problems.
  • It prepares them for later work with arrays and datasets.
  • It reduces syntax confusion early.

Projects list for this week

Calculator Builder Create a simple arithmetic calculator with input, output, and operator handling.
Student Marks Summary Take subject marks, compute total, average, and a simple performance message.
Unit Converter Build temperature, length, or weight converters to strengthen numeric logic.

Week 5: Statistics for AI

Students learn how to describe and interpret data properly. The emphasis is not only on formulas, but on meaning: central tendency, spread, position, and variation.

Why this week matters

  • It builds stronger reasoning for later ML evaluation.
