Week 1 Assignment

Python Foundations I — Assignment Pack

Use these tasks for classwork, homework, lab practice, or portfolio evidence.

Tasks

  1. Write a program that reads two numbers and prints sum, difference, product, quotient, and remainder.
  2. Take a student's name and marks in three subjects, then print total and average.
  3. Create a unit converter for kilometers to meters and centimeters.

Submission checklist

  • Clean code with comments
  • Input and output samples
  • One short explanation paragraph
  • One reflection: what was easy and what was difficult

Teacher note

Encourage students to explain their reasoning. Even in beginner weeks, clarity matters as much as code correctness.
