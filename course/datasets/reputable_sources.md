# Reputable Data Sources

## General
- Kaggle
- UCI Machine Learning Repository
- data.gov
- World Bank Open Data
- WHO public datasets
- IMD / NOAA / open weather archives where permitted

## Finance
- Yahoo Finance for educational exploration
- Official exchange and company disclosures

## Sports
- Official league data, score feeds, or cleaned educational datasets

## Good practice
- Check license
- Check update frequency
- Read metadata
- Avoid scraping that violates terms
- Never treat noisy domains like finance as easy prediction problems
