1. What is Statistics?

Statistics is the study of data. It helps us collect data, organize it, summarize it, understand patterns, compare groups, measure uncertainty, and make decisions.

In AI and Machine Learning, data is the raw material. Statistics is the set of tools that helps us inspect that raw material before we trust it.

Simple picture

Data tells a story, but usually in a scattered and noisy form. Statistics helps us read that story clearly.

2. Why Statistics Matters in AI-ML

Understand the center

We need to know what is typical in the dataset.

Understand the spread

We need to know whether values are tightly packed or scattered.

Detect bad values

Outliers and strange values can damage a model.

Compare features

Some features may be more useful than others.

Scale data correctly

Normalization and standardization depend on statistics.

Reason with uncertainty

Real data is messy. Statistics gives discipline to our thinking.

5.1 Mean

Center of the data
Intuition

Mean tells us the average or central value. It answers the question: if the total were shared equally, how much would each value get?

Formula

Mean = (sum of all values) / (number of values)

AI-ML Use

Mean is used to summarize features, center data, and standardize values before training.

Hand Calculation

Dataset: 2, 4, 4, 4, 5, 5, 7, 9

Sum = 2 + 4 + 4 + 4 + 5 + 5 + 7 + 9 = 40

Count = 8

Mean = 40 / 8 = 5

Important Thought

Mean is sensitive to outliers. A very large or very small value can pull it away from the true center.

data = [2, 4, 4, 4, 5, 5, 7, 9] total = 0 for x in data: total += x mean = total / len(data) print("Mean:", mean)
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) print("Mean:", np.mean(data))

5.2 Median

Middle of the sorted data
Intuition

Median is the middle value after sorting. It is often better than mean when extreme values are present.

Formula Idea

Sort first. If the number of values is odd, take the middle one. If it is even, average the two middle values.

AI-ML Use

Median is useful in robust summaries and in filling missing values when outliers exist.

Hand Calculation

Sorted data: 2, 4, 4, 4, 5, 5, 7, 9

There are 8 values, so we take the 4th and 5th values.

4th = 4

5th = 5

Median = (4 + 5) / 2 = 4.5

Why median matters

If one value were 900 instead of 9, the mean would jump a lot, but the median would stay much more stable.

data = [2, 4, 4, 4, 5, 5, 7, 9] sorted_data = sorted(data) n = len(sorted_data) if n % 2 == 0: median = (sorted_data[n//2 - 1] + sorted_data[n//2]) / 2 else: median = sorted_data[n//2] print("Median:", median)
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) print("Median:", np.median(data))

5.3 Variance

Spread from the mean
Intuition

Variance measures how far the values are from the mean, on average, after squaring the distances.

Formula

Variance = average of (value - mean)2

AI-ML Use

Variance helps us understand stability, spread, feature scale, and noise in a dataset.

Hand Calculation

Mean = 5

Differences: -3, -1, -1, -1, 0, 0, 2, 4

Squared differences: 9, 1, 1, 1, 0, 0, 4, 16

Sum of squared differences = 32

Variance = 32 / 8 = 4

Why square the distances?

Because negative and positive distances would otherwise cancel each other. Squaring also gives more weight to large deviations.

data = [2, 4, 4, 4, 5, 5, 7, 9] mean = sum(data) / len(data) sq_sum = 0 for x in data: sq_sum += (x - mean) ** 2 variance = sq_sum / len(data) print("Variance:", variance)
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) print("Variance:", np.var(data))
Mini Quiz

5.4 Standard Deviation

Spread in original units
Intuition

Standard deviation is a more interpretable version of variance because it comes back to the original unit.

Formula

Standard Deviation = √Variance

AI-ML Use

Standard deviation is heavily used in standardization, anomaly detection, and distribution understanding.

Hand Calculation

Variance = 4

Standard Deviation = √4 = 2

Interpretation

A larger standard deviation means the values are more spread out. A smaller one means the data is tightly packed.

data = [2, 4, 4, 4, 5, 5, 7, 9] mean = sum(data) / len(data) sq_sum = 0 for x in data: sq_sum += (x - mean) ** 2 variance = sq_sum / len(data) std = variance ** 0.5 print("Standard Deviation:", std)
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) print("Standard Deviation:", np.std(data))

5.5 Minimum, Maximum, and Range

Boundary values
Intuition

Minimum and maximum tell us the smallest and largest values. Range tells us the full spread from lowest to highest.

Formula

Range = Maximum - Minimum

AI-ML Use

Helps with scaling, spotting suspicious values, and understanding data limits.

Hand Calculation

Dataset: 2, 4, 4, 4, 5, 5, 7, 9

Minimum = 2

Maximum = 9

Range = 9 - 2 = 7

Limitation

Range depends only on two values. It ignores everything in the middle, so it is not enough by itself.

data = [2, 4, 4, 4, 5, 5, 7, 9] minimum = min(data) maximum = max(data) data_range = maximum - minimum print("Minimum:", minimum) print("Maximum:", maximum) print("Range:", data_range)
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) print("Minimum:", np.min(data)) print("Maximum:", np.max(data)) print("Range:", np.max(data) - np.min(data))

5.6 Percentiles

Relative position in data
Intuition

Percentiles tell us how a value stands relative to the whole dataset. They divide data into percentage positions.

Main idea

50th percentile is the median. 25th and 75th percentiles show lower and upper quarters.

AI-ML Use

Useful for thresholds, capping outliers, quartile analysis, and robust scaling.

Direct Understanding

Sorted data: 2, 4, 4, 4, 5, 5, 7, 9

25th percentile lies in the lower quarter of the data.

50th percentile is the median.

75th percentile lies in the upper quarter of the data.

Important Note

Exact percentile calculation can differ slightly depending on interpolation method. Software handles this consistently.

# Percentiles are usually better computed using libraries, # but the main idea is to sort the data and find the # required relative position in the ordered list.
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) print("25th:", np.percentile(data, 25)) print("50th:", np.percentile(data, 50)) print("75th:", np.percentile(data, 75))

5.7 Correlation

How two variables move together
Intuition

Correlation measures whether two variables tend to rise together, fall together, or move independently.

Interpretation

Close to 1 means strong positive relation. Close to -1 means strong negative relation. Close to 0 means weak linear relation.

AI-ML Use

Used in feature selection, redundancy detection, and early exploration of relationships with the target.

Direct Understanding

Hours: 1, 2, 3, 4, 5

Marks: 20, 35, 45, 60, 75

As hours increase, marks increase.

This suggests positive correlation.

Very Important

Correlation does not prove causation. Two things moving together does not always mean one causes the other.

# Full hand coding of correlation is longer because it uses: # 1. mean of x # 2. mean of y # 3. deviations # 4. product of deviations # 5. standard deviations of both lists # For learning, focus first on interpretation.
import numpy as np hours = np.array([1, 2, 3, 4, 5]) marks = np.array([20, 35, 45, 60, 75]) corr_matrix = np.corrcoef(hours, marks) print(corr_matrix) print("Correlation:", corr_matrix[0, 1])

8. Normalization and Standardization

Normalization

Normalization often means scaling values into a fixed range such as 0 to 1.

value_normalized = (x - min_value) / (max_value - min_value)

Standardization

Standardization shifts the feature to mean 0 and standard deviation 1.

z = (x - mean) / std
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) normalized = (data - np.min(data)) / (np.max(data) - np.min(data)) print(normalized)
import numpy as np data = np.array([2, 4, 4, 4, 5, 5, 7, 9]) standardized = (data - np.mean(data)) / np.std(data) print(standardized)
Why does this matter in ML?

Some models behave much better when feature scales are comparable. Large-scale features can dominate smaller ones if we do not scale properly.

9. How These Ideas Connect to AI-ML

Regression

We inspect center, spread, and relation between input and target.

Classification

We inspect class balance, per-feature behavior, and noise.

Feature Selection

Correlation and spread can reveal useful or redundant features.

Outlier Handling

Range, median, quartiles, and spread can reveal suspicious points.

Preprocessing

Standardization and normalization depend directly on mean, min, max, and standard deviation.

Evaluation

Statistical thinking helps interpret errors, residuals, and model stability.

10. One Worked ML-Flavored Example

Suppose you want to predict marks from hours studied.

Hours1,2,3,4,5,6
Marks20,30,42,55,68,80
NeedMean
NeedStd Dev
NeedCorrelation
import numpy as np hours = np.array([1, 2, 3, 4, 5, 6]) marks = np.array([20, 30, 42, 55, 68, 80]) print("Average hours:", np.mean(hours)) print("Average marks:", np.mean(marks)) print("Std of marks:", np.std(marks)) print("Min hours:", np.min(hours)) print("Max hours:", np.max(hours)) print("Correlation matrix:") print(np.corrcoef(hours, marks))
Why do this before training?

Because if the relation is weak, the spread is wild, or the data contains extreme problems, model training becomes less meaningful.
