Level 0 explanation

A word is not only spelling

Spelling matching

Spelling matching checks whether two words look exactly the same. For example, happy and happy match.

happy = happy

Meaning matching

Meaning matching checks whether two words carry a similar idea. For example, happy and glad are different spellings, but their meanings are close.

happy ≈ glad
Main idea: exact matching checks spelling, but NLP should also try to match meaning.

51%

What is word meaning?

A word can have spelling, sound, meaning, usage, and relationships with other words.

Part Example NLP question
Spelling fast Which letters are used?
Meaning quick What idea does the word carry?
Usage The car is fast. How is the word used in context?
Other meaning He is on a fast. Does the same spelling mean something else?

The word fast can mean quick, but it can also mean not eating food. This is why NLP needs context.

52% and 55%

Synonyms: matching similar meanings

What are synonyms?

Synonyms are words with similar meanings.

  • big = large
  • small = little
  • happy = glad
  • quick = fast
  • smart = intelligent
  • begin = start
  • end = finish
  • buy = purchase

Context warning

Synonyms are not always exactly equal in every sentence.

Good big house ≈ large house
Awkward big brother ≠ large brother

Beginner Python: synonym finder

synonyms = {
    "happy": ["glad", "joyful", "pleased"],
    "sad": ["unhappy", "sorrowful"],
    "big": ["large", "huge"],
    "small": ["little", "tiny"],
    "fast": ["quick", "rapid"],
    "start": ["begin", "commence"],
    "end": ["finish", "complete"]
}

word = input("Enter a word: ").lower()

if word in synonyms:
    print("Synonyms of", word, "are:")
    for s in synonyms[word]:
        print("-", s)
else:
    print("Sorry, word not found.")

53% and 56%

Antonyms: matching opposite meanings

What are antonyms?

Antonyms are words with opposite meanings.

  • hot ↔ cold
  • big ↔ small
  • happy ↔ sad
  • fast ↔ slow
  • start ↔ stop
  • open ↔ close
  • true ↔ false
  • success ↔ failure

Meaning can depend on sentence

light bag means not heavy.

light room means bright room.

Turn on the light means lamp or brightness.

The same word can change meaning depending on context.

Beginner Python: antonym finder

Matching similar meanings in sentences

Similar meaning

Sentence 1: I am happy today.

Opposite meaning

Sentence 1: The box is big.

Sentence 2: The box is small.

big ↔ small

Python: word-level similarity check

synonyms = {
    "happy": ["glad", "joyful"],
    "sad": ["unhappy", "sorrowful"],
    "big": ["large", "huge"],
    "small": ["little", "tiny"],
    "fast": ["quick", "rapid"],
    "start": ["begin"],
    "end": ["finish"]
}

def are_words_similar(word1, word2):
    word1 = word1.lower()
    word2 = word2.lower()

    if word1 == word2:
        return True

    if word1 in synonyms and word2 in synonyms[word1]:
        return True

    if word2 in synonyms and word1 in synonyms[word2]:
        return True

    return False

print(are_words_similar("happy", "glad"))
print(are_words_similar("big", "large"))
print(are_words_similar("happy", "sad"))

Problem with CountVectorizer and TF-IDF

CountVectorizer and TF-IDF are useful, but they mostly depend on visible words. They may not understand that different words can have similar meanings.

Sentence 1

I am happy.

Sentence 2

I am glad.

Meaning

happy ≈ glad

Better meaning knowledge may come from synonym dictionaries, antonym dictionaries, WordNet, word embeddings, sentence embeddings, and transformer models. In this lesson, we keep it beginner-friendly with dictionaries.

Project 2: Meaning Matcher for Sentences

word_data = {
    "happy": {
        "synonyms": ["glad", "joyful", "pleased"],
        "antonyms": ["sad", "unhappy"]
    },
    "sad": {
        "synonyms": ["unhappy", "sorrowful"],
        "antonyms": ["happy", "glad"]
    },
    "big": {
        "synonyms": ["large", "huge", "giant"],
        "antonyms": ["small", "little", "tiny"]
    },
    "small": {
        "synonyms": ["little", "tiny"],
        "antonyms": ["big", "large", "huge"]
    },
    "fast": {
        "synonyms": ["quick", "rapid", "speedy"],
        "antonyms": ["slow"]
    },
    "slow": {
        "synonyms": ["lazy", "sluggish"],
        "antonyms": ["fast", "quick", "rapid"]
    },
    "start": {
        "synonyms": ["begin", "commence"],
        "antonyms": ["stop", "end", "finish"]
    },
    "open": {
        "synonyms": ["unlock"],
        "antonyms": ["close", "shut"]
    }
}

def clean_sentence(sentence):
    sentence = sentence.lower()
    marks = [".", ",", "!", "?", ";", ":"]
    for mark in marks:
        sentence = sentence.replace(mark, "")
    return sentence.split()

def are_synonyms(word1, word2):
    if word1 == word2:
        return True

    if word1 in word_data and word2 in word_data[word1]["synonyms"]:
        return True

    if word2 in word_data and word1 in word_data[word2]["synonyms"]:
        return True

    return False

def are_antonyms(word1, word2):
    if word1 in word_data and word2 in word_data[word1]["antonyms"]:
        return True

    if word2 in word_data and word1 in word_data[word2]["antonyms"]:
        return True

    return False

def meaning_match(sentence1, sentence2):
    words1 = clean_sentence(sentence1)
    words2 = clean_sentence(sentence2)

    exact_matches = 0
    synonym_matches = 0
    antonym_conflicts = 0

    for w1 in words1:
        for w2 in words2:
            if w1 == w2:
                exact_matches += 1
                break
            elif are_synonyms(w1, w2):
                synonym_matches += 1
                break
            elif are_antonyms(w1, w2):
                antonym_conflicts += 1
                break

    meaning_score = exact_matches + synonym_matches - antonym_conflicts
    return exact_matches, synonym_matches, antonym_conflicts, meaning_score

s1 = input("Enter first sentence: ")
s2 = input("Enter second sentence: ")

exact, synonym, antonym, score = meaning_match(s1, s2)

print("\nExact matches:", exact)
print("Synonym matches:", synonym)
print("Antonym conflicts:", antonym)
print("Final meaning score:", score)

if score > 2:
    print("Result: Sentences are meaningfully similar.")
elif score == 0:
    print("Result: Sentences are weakly related or unclear.")
elif score < 0:
    print("Result: Sentences may have opposite meaning.")
else:
    print("Result: Sentences have some similarity.")

Revision Test

What is a synonym?

A synonym is a word with similar meaning, such as happy and glad.

What is an antonym?

An antonym is a word with opposite meaning, such as hot and cold.

Why is exact matching not enough?

Because different words can express similar meanings, such as begin and start.

Why can TF-IDF fail with synonyms?

Because TF-IDF sees happy and glad as different terms unless extra meaning knowledge is added.

What is an antonym conflict?

An antonym conflict happens when one sentence contains a word and another sentence contains its opposite.

Why is the word light confusing?

It can mean not heavy, brightness, or a lamp depending on context.
